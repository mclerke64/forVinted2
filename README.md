# React Color Picker (Mini Project)

A simple, interactive React application created during my early learning stages to understand the core principles of React state management and dynamic UI rendering.

## 🎯 Project Goals & Features
The main focus of this project was to practice handling user input and binding it directly to the user interface.
* **State Management:** Uses the React `useState` hook to dynamically track the chosen color value.
* **Event Handling:** Utilizes an `onChange` listener to capture real-time values from a native HTML color input.
* **Dynamic Styling:** Implements React inline styles (`style={{ backgroundColor: ... }}`) to instantly update a visual element container based on user preference.

## 🛠️ Tech Stack
* **React**
* **JavaScript (ES6+)**
* **HTML5 & Inline CSS**

## 💡 What I Learned
Through this project, I mastered how state flows in React, how to create controlled components, and how to change the appearance of components dynamically without reloading the page. It serves as a solid foundation for my transition into more advanced React topics, such as working with custom hooks, external libraries, and TypeScript.
